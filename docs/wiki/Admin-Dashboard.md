# Admin Dashboard

The admin dashboard is the central hub for chapter leads to manage their chapter and events.

---

## Accessing the Dashboard

Click **Dashboard** in the navigation bar (visible only to admin users).

![Admin Dashboard](screenshots/20-admin-dashboard.png)

---

## Dashboard Sections

The dashboard has three main sections, accessible via the buttons at the top:

![Dashboard Navigation](screenshots/10b-dashboard-nav.png)

### 📋 Your Events

The default view shows all events for your chapter. Each event card displays:
- Event title
- Date and location
- Registration count

Click on an event card to manage it — see [Managing Attendees](Managing-Attendees).

### ➕ Create Event

Opens the event creation form — see [Creating Events](Creating-Events).

### ✏️ Edit Chapter

Opens the chapter editor — see [Editing Your Chapter](Editing-Your-Chapter).

---

## Event Detail View

Click on any event card to see the full attendee list and management tools.

![Event Detail](screenshots/23-event-detail.png)

From here you can:
- View all registered attendees with their roles
- Check in attendees
- Update attendee roles (attendee → volunteer → speaker → sponsor)
- Export attendance as CSV
- Issue event badges
- Manually register walk-in attendees
